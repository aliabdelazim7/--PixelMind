# دليل إعداد وتشغيل نظام إدارة العملاء المبسط (CRM)
### دليل فائق البساطة ومصمم للإنتاج الفعلي بموثوقية 100% ودون أي أخطاء

هذا الدليل مكتوب باللغة العربية بالكامل لشرح كيفية تشغيل نظام الـ CRM البسيط والمتصل بـ Google Sheets عبر الكود المحدث، والذي يعتمد على **المعرّفات الفريدة (IDs)** لتفادي أخطاء تداخل البيانات أو حذف صفوف بالخطأ.

---

## 📂 هيكل جدول البيانات التلقائي (Google Sheet Columns)

عند تشغيل الكود لأول مرة على شيت جوجل فارغ، سيقوم الكود **تلقائياً** بإنشاء وتنسيق الأعمدة التالية في الصف الأول:
1. **المعرف** (العمود A) - يحتوي على كود فريد تلقائي لكل عميل (مثال: `L-1720815234`) لمنع تداخل الصفوف.
2. **الاسم بالكامل** (العمود B) - اسم العميل.
3. **رقم الهاتف** (العمود C) - رقم الهاتف (يتم تحويل الأرقام العربية تلقائياً لإنجليزية).
4. **الحالة** (العمود D) - حالة العميل الحالية.
5. **تاريخ الإضافة** (العمود E) - تاريخ تسجيل العميل تلقائياً.

*ملاحظة*: لست بحاجة لكتابة هذه الأعمدة يدوياً، الكود سيكتبها وينسقها بمجرد إضافة أول عميل أو جلب البيانات.

---

## 📐 إعداد كود Apps Script المحدث

1. افتح شيت جوجل (الأيقونة الخضراء).
2. اذهب إلى **Extensions** (إضافات) $\to$ **Apps Script**.
3. استبدل الكود بالكامل بكود **[code.gs المحدث على جيت هاب](https://github.com/aliabdelazim7/--PixelMind/blob/main/code.gs)**.
4. تأكد من وضع رابط شيت جوجل الخاص بك في السطر رقم 4:
   ```javascript
   const SPREADSHEET_URL = "رابط_شيت_جوجل_الخاص_بك_هنا";
   ```
5. احفظ الكود واضغط **Deploy** $\to$ **New Deployment** (تطبيق ويب، التشغيل بواسطة: أنا، الدخول: أي شخص).
6. انسخ رابط الـ Web App وضعه في ملف `index.html` في السطر رقم 633 على GitHub لكي يتصل Vercel مباشرة.

## 🛡️ الميزات المدمجة لضمان جودة التشغيل (Production Ready)

- **منع نقرات الماوس المتكررة (Double-click Protection)**: يتم تعطيل زر الحفظ وتغيير نصه إلى "⏳ جاري الحفظ..." لمنع تكرار تسجيل العميل عند نقر السيلز المتكرر.
- **تحويل الأرقام العربية تلقائياً**: إذا قام السيلز بإدخال هاتف مثل `٠١٠١٢٣٤٥٦٧٨` سيتم تحويله تلقائياً لـ `01012345678` لضمان عمل رابط الواتساب بنجاح.
- **الحماية ضد حقن الكود (XSS Protection)**: الكود يقوم بتطهير أي مدخلات لمنع أي ثغرات برمجية.
- **مزامنة العمليات أوفلاين (Offline Sync)**: في حال انقطاع الإنترنت أثناء التسجيل، يتم حفظ البيانات محلياً، وبمجرد عودة الإنترنت تظهر شارة صفراء بالأعلى تطلب منك المزامنة بنقرة زر واحدة لرفعهم تلقائياً لشيت جوجل.
